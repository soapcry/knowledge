# Union Find - Система непересекающихся множеств (дисджойнт-сет)

```java
public class UnionFind {

    private int[] root;

    // Конструктор инициализирует структуру данных для заданного количества
    // элементов. Каждый элемент начинается в своем собственном множестве,
    // поэтому каждое значение массива устанавливается равным его индексу.

    public UnionFind(int size) {
        root = new int[size];
        for (int i = 0; i < size; i++) {
            root[i] = i;
        }
    }

    // Метод find возвращает корень (представителя) множества, к которому принадлежит
    // элемент x. В данной реализации это просто значение элемента массива - root[x].

    public int findRoot(int x) {
        return root[x];
    }

    // Метод union объединяет множества, к которым принадлежат элементы x и y.
    // Сначала определяются представители этих множеств. Если они разные, то
    // алгоритм обходит все элементы массива и заменяет все вхождения одного
    // представителя на другого, тем самым объединяя множества под одним.

    public void union(int x, int y) {
        int root_x = findRoot(x);
        int root_y = findRoot(y);

        if (root_x != root_y) {
            for (int i = 0; i < root.length; i++) {
                if (root[i] == root_y) {
                    root[i] == root_x;
                }
            }
        }
    }

    // Метод connected проверяет, принадлежат ли элементы x и y одному и тому же
    // множеству, сравнивая их представителей в массиве root.

    public boolean connected(int x, int y) {
        return findRoot(x) == findRoot(y);
    }
}
```
